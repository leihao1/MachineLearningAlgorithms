# Item-Item Collaborative Filtering tool
Simple tool to predict item rating will be given by specific users
with Collaborative Filtering algorithsm.

Function `sim()`
-
>calculate cosine similarity value between two vectors
>
> `>>>a=[1,2,3]`
>
>`>>>b=[3,2,1]`
>
>`>>>sim(a,b)`
>
>`0.7142857142857143`
